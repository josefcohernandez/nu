§3 · Lua puede hacer cualquier cosa

No hay «zona privilegiada» reservada al core. El usuario que quiere un harness distinto al oficial no hace fork: escribe Lua.

Nota al margen — la única excepción deliberada son los providers de LLM, que se declaran como datos (TOML), no como código. Ver ADR-005.

Filosofía de nu

El proyecto no es el agente: es un kernel mínimo y un sistema de extensiones donde todo lo demás — incluido el agente — es una extensión.

LA VARA PARA CASOS DUDOSOS

Si algo se describe con vocabulario del kernel (plugins, rutas, versiones), es del kernel. Si necesita vocabulario de producto (agente, chat, tools), es de una extensión.

6. La API del core es sagrada

Si todo se construye sobre las primitivas, romperlas rompe el mundo. La API v1 debe ser deliberadamente pequeña y aburrida, y crecer solo por adición.
